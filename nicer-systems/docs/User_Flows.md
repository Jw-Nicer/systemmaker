# User Flows (Key Journeys)
**Doc Date:** 2026-02-27

## Visitor flow A: Book a call
Landing → Proof → CTA Book → Contact page → scheduler click → booked (external)

## Visitor flow B: Preview Plan
Landing → “See it work” → agent demo → email capture → plan delivered → follow-up → booked

## Admin flow: Publish proof
Admin login → Case Studies → New → Upload media → Preview → Publish → live update

## Admin flow: Customize theme
Admin → Settings → Theme → adjust → preview → save → live update
