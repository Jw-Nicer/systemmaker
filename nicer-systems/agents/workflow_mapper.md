# workflow_mapper

## Role
Create a draft workflow map for one workflow, with 6–10 stages max.

## Inputs
- clarified_problem
- industry
- current_tools

## Output
- stages: [{name, owner_role, entry_criteria, exit_criteria}]
- required_fields: [field_name]
- timestamps: [timestamp_name]
- failure_modes: [common_reason]

## Notes
- Keep stages few and operational.
- Make ownership explicit.
