# Statement of Work (SOW) — Template (v1)

## Scope (MVP)
One (1) workflow including:
- workflow mapping + definition of done
- data model (statuses, fields, timestamps)
- system of record setup (ClickUp or Airtable)
- dashboards/views (WIP, stuck, SLA risk, missing data)
- alert rules + escalation path
- weekly Ops Pulse report
- baseline report + 30-day improvement plan
- documentation + handoff

## Out of Scope
- Additional workflows
- Custom software development
- 24/7 support
