# Discovery Script + Intake (v1)

## Goal of the call (15–45 minutes)
Lock:
1) One workflow to fix  
2) One output definition (“done”)  
3) One internal owner  
4) One system of record tool  

## Script (Facilitator prompts)
### A) Context (5 min)
- What’s the workflow that creates the most “Where is this?” chatter?
- What happens if it’s delayed?

### B) Define the workflow (10–15 min)
- What triggers the work to start?
- What are the stages from start → done?
- Who owns each stage (role or person)?
- What’s the definition of “done”?

### C) Visibility pain (5–10 min)
- How do you track status today?
- Where do updates get lost?
- What are the top 3 reasons things get stuck?

### D) Data + tools (5–10 min)
- Current tools (ClickUp/Airtable/Sheets/Email/Slack)?
- Where does intake come from (forms, email, phone)?
- Do you have SLAs or deadlines?

### E) Success metrics (5 min)
Pick 2–3:
- reduce overdue items
- reduce status-chasing messages
- reduce cycle time
- reduce rework / reopen rate
- improve weekly reporting reliability

### F) Scope lock (2 min)
**Confirm:** MVP includes **one workflow only**.  
Any second workflow becomes Phase 2.
